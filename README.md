🇨🇴 Gastronomía de Colombia - Smart Contract (Base Mainnet)
Este es el undécimo contrato inteligente de mi colección profesional "Sabores de América", y el primero de la etapa de Sudamérica. Colombia es un país de regiones, desde las cumbres de los Andes hasta las costas del Caribe y el Pacífico. Este contrato introduce la variable técnica de Acompañante, capturando la esencia de la "pareja ideal" de cada plato colombiano.

El contrato marca el inicio de la expansión por el cono sur, manteniendo una arquitectura de seguridad robusta y gas-efficient.

🛠️ Detalles Técnicos: Sabor Regional On-Chain

Este despliegue consolida las mejores prácticas de desarrollo en Base:

Hito Sudamericano: Primer bloque de la serie en territorio suramericano, demostrando la escalabilidad del proyecto culinario-técnico.

Metadata de Acompañante (string acompanante): Variable diseñada para documentar el escolta tradicional de cada plato (ej. Arepa de Maíz, Arroz de Coco, Patacón Pisao), una distinción cultural profunda entre las regiones de Colombia.

Seguridad Auditada: Lógica sin manejo de fondos para eliminar vectores de ataque financieros, optimizada para la capa 2 de Ethereum.

Estándar SPDX-MIT: Transparencia total y cumplimiento de licencias para un portafolio profesional de nivel senior.

📖 Modo de Uso

Explorar la Sabrosura: En la pestaña Read Contract, usa consultarPlato (ID 1 para la Bandeja Paisa) para ver el acompañante registrado.

Registrar Tradición: En la pestaña Write Contract, usa registrarPlato para añadir joyas como "Ajiaco Santafereño" (Acompañante: Alcaparras y Crema) o "Pescado Frito" (Acompañante: Arroz de Coco).

Gobernanza Culinaria: Vota por tus platos favoritos mediante el sistema inmutable de Likes y Dislikes.
